# 대전략 시뮬레이션 — 프로토타입

> CK3 가문 + HOI4 사단편성 + Victoria3 경제 + 문명 교역
> HTML Canvas + JavaScript 프로토타입 → Unity 본개발 예정

---

## 빠른 시작

```
패러독스 게임 바이브 코딩/
├── index.html      ← 게임 실행 (브라우저에서 열기)
├── editor.html     ← 맵 에디터 (프로빈스 직접 제작)
├── progress.html   ← 개발 진척도 대시보드
```

**별도 서버 불필요** — 파일을 브라우저에서 직접 열면 됩니다.

---

## 조작 방법

| 입력 | 동작 |
|------|------|
| 좌클릭 | 프로빈스 선택 |
| 스크롤 | 맵 줌 (0.5×~3.0×) |
| 우클릭 드래그 | 맵 패닝 |
| `Space` | 일시정지/재개 |
| `1` `2` `3` `5` | 배속 전환 |
| `F` | 세력 목록 패널 |

## 게임 개요

### 핵심 설계 철학

1. **경제 비개입** — 플레이어는 경제에 직접 개입 불가. 전략적 선택이 간접적으로 경제력을 결정.
2. **인구 = 최고 자원** — 자연성장 없음. 이벤트/정복으로만 획득. 전쟁 비용이 극도로 높음.
3. **이벤트 중심** — 콘텐츠 대부분이 이벤트/디시전. JSON 데이터 드리븐.
4. **체제의 연속** — CK3식 혈통이 아닌, 중국 왕조 교체식 계승. 후계자는 무엇이든 가능.
5. **영향권 정치** — 관계도 기반 유지 가문. 관계가 곧 국력.

## 맵 에디터 사용법

`editor.html`을 열면 맵 에디터가 실행됩니다.

### 자동 감지 (단색 PNG 권장)

1. **📂 이미지 불러오기** → 각 프로빈스를 다른 단색으로 칠한 PNG 선택
2. **🔍 자동 감지** → 색상별 영역을 자동으로 폴리곤으로 변환
3. 속성 편집 (이름, 세력, 지형, 자원, 건물)
4. **⬇ JSON 내보내기** → `PROVINCES_DATA`를 복사해서 `index.html`에 붙여넣기

### 수동 그리기

- `D` 그리기 / `S` 선택 / `E` 삭제
- 좌클릭으로 꼭짓점 추가, 더블클릭으로 폴리곤 완성
- `G` 격자 스냅 / `F` 화면 맞춤 / `Z` 되돌리기
